03 —

如何使用

STEP 01
预览对比
点击任一方案卡片的「▶ 在此预览」按钮，在本页内 iframe 中实时查看，无需跳转。可在预览内切换该方案的主题。
STEP 02
独立打开
「↗ 新窗口打开」会在新标签页打开原始 HTML，获得完整交互体验（路由动画、滚动进度、主题切换）。
STEP 03
迁移到 React
选定方向后，参考 10-ysrouter-design-spec.md §10.7 的迁移路径，落地到 features/home/。
