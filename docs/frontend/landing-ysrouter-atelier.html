DEMONSTRATION § II

Swap one URL. That's it.

Point any OpenAI-compatible SDK at YSRouter. Routing, keys, billing, and observability are handled — transparently, in the margins.

~/app — bash fig. ii.a
# before — direct to one provider
$ curl https://api.openai.com/v1/chat/completions \
   -H "Authorization: Bearer sk-..."

# after — every model, one signature
$ curl https://api.ysrouter.com/v1/chat/completions \
   -H "Authorization: Bearer sk-ys-..." \
   -d '{"model":"claude-sonnet-4","messages":[{"role":"user","content":"hi"}]}'
